# book-release-it

Release It! (2nd) 학습 정리 프로젝트.

## 구성
- `maps/ch01.md` ~ `maps/ch20.md`: 챕터별 핵심 요약 + 실무 적용 액션
- `adrs/`: 도입 결정 기록
- `interviews/`: 개념 검증용 Q&A

## 사용 방법
1. 챕터 1개를 읽고 `maps/chXX.md`를 확인
2. 실무 적용 1개를 선택해 이슈로 등록
3. 완료 후 회고를 남기고 다음 챕터 진행

## 권장 학습 리듬
- 주 2챕터(총 10주)
- 매주 1회 GameDay/런북 점검
- 월말에 SLO/MTTR 개선 수치 확인
