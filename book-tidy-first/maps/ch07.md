# Ch07. Move Declaration and Initialization Together

## TL;DR
- 변수 선언과 초기화는 가능한 한 가까이 두라.
- 선언과 사용 사이의 거리가 멀수록 코드를 이해하기 어렵다.

## Key Ideas
- 변수의 스코프를 최소화
- "이 변수가 뭐였지?" 하며 위로 스크롤하는 일을 줄인다
- 필요한 시점에 선언하고 바로 초기화

## Open Questions
- 언어별 제약이 있을 때는?
  - C 같은 언어는 함수 시작부에 선언해야 할 수 있다
  - 그래도 초기화는 사용 직전에 하도록 분리 가능
