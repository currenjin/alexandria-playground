# Ch12. Extract Helper

## TL;DR
- 반복되거나 복잡한 로직을 헬퍼 함수로 추출하라.
- 좋은 이름의 헬퍼 함수는 코드의 의도를 명확히 한다.

## Key Ideas
- Helper Function: 특정 작업을 수행하는 작은 함수
- 추상화 수준을 맞춘다 (한 함수 내에서 추상화 수준 일관성)
- "어떻게"보다 "무엇을"이 드러나게 한다

## Open Questions
- 헬퍼 함수의 적절한 크기는?
  - 한 가지 일만 하는 것
  - 이름으로 설명할 수 있는 것
  - 너무 작으면 오히려 흐름을 끊을 수 있다
