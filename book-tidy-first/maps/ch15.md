# Ch15. Delete Redundant Comments

## TL;DR
- 코드가 이미 말하고 있는 것을 반복하는 주석은 삭제하라.
- 중복 주석은 유지보수 부담이고 거짓말이 될 수 있다.

## Key Ideas
- 코드와 주석이 같은 말을 하면 주석은 불필요
- `i++; // i를 1 증가` 같은 주석
- 코드가 변경되면 주석이 거짓이 된다

## Open Questions
- 주석을 지우기 전에 확인할 것은?
  - 주석이 정말로 중복인지 (미묘한 뉘앙스가 있을 수 있다)
  - 지운 후에도 코드가 명확한지
  - 필요하다면 코드 자체를 더 명확하게 수정
