# plan.md - String Calculator (Kotlin)

## 빈 문자열과 단일 숫자
- [x] 빈 문자열 "" -> 0 반환
- [x] 숫자 하나 "1" -> 해당 숫자 반환

## 두 개의 숫자
- [x] 쉼표로 구분된 두 숫자 "1,2" -> 합계 3 반환

## 임의 개수의 숫자
- [x] "1,2,3,4,5" -> 합계 15 반환

## 줄바꿈 구분자 허용
- [x] 쉼표 외에 줄바꿈(\n)도 구분자로 허용
- [x] "1\n2,3" -> 6 반환

## 음수 예외 처리
- [x] 음수가 포함되면 예외 발생
- [] 예외 메시지에 음수 값 포함: "negatives not allowed: -1, -3"

## 1000 초과 숫자 무시
- [] "2,1001" -> 2 반환

## 임의 길이 구분자
- [] //[***]\n1***2***3 -> 6 반환

## 다중 구분자
- [] //[*][%]\n1*2%3 -> 6반환
